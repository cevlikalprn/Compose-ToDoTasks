<h2> Compose-ToDoTasks </h2>
A To-Do App demo written with Jetpack Compose

<h2> Tech Stack </h2>

* 100% Kotlin
* Jetpack Compose
* Material Design
* Coroutines + Flows
* Hilt for Dependency Injection
* Room for Local Database 
* MVVM + Clean Architecture
